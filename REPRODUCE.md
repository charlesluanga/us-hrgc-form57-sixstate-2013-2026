# Reproduce headline results

Run from **repository root** (folder containing `requirements.txt` and `data/`).

## Environment

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Step 1 — Cohort (skip if using bundled `data/all_states.csv`)

1. Download FRA Form 57 CSV to repository root.
2. Run:

```bash
python scripts/clean_hrgc.py --source csv
```

**Check:** `data/cohort_manifest.json` → `"n_rows": 8394`

## Step 2 — Primary analyses

```bash
python scripts/run_revision_analyses.py
```

**Output:** `outputs/revision_results.json`

## Step 3 — Figures (optional; scripts only in repo)

Reviewers **regenerate** panels locally (PNG written to `outputs/figures/`). The repository does **not** store figure binaries.

```bash
python scripts/build_manuscript_figures.py
python scripts/run_treeshap_analysis.py
```

**Not shared:** conceptual Figure 1 (author-designed; not generated by these scripts).

## Step 4 — Verify

```bash
python scripts/verify_replication.py
```

## Runtime

Primary analyses: ~5–15 minutes on a typical laptop. Full figure build: ~3–5 minutes additional.
